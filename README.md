Este proyecto corresponde al sistema web de gestión interna de la Clínica Psicológica Reconstruye.

La plataforma permite administrar usuarios, gestionar citas psicológicas y organizar los procesos internos de la clínica mediante distintos niveles de acceso según el rol del usuario.

Objetivos del sistema:

-Digitalizar y optimizar la gestión de:
-Registro y autenticación de usuarios
-Agendamiento de citas psicológicas
-Administración de pacientes y psicólogos
-Control total por parte del administrador
